## Quick Reference

### Algorithms by Category

**Regression**
- Linear Regression (`/algorithms` card 1)
  - Formula: y = mx + b
  - Use: Predict continuous values
  
**Classification**
- Logistic Regression (`/algorithms` card 2)
  - Formula: P(y=1) = sigmoid(z)
  - Use: Binary/multi-class classification
  
- Decision Trees (`/algorithms` card 3)
  - Formula: Information Gain
  - Use: Interpretable decisions

- Random Forest (`/algorithms` card 4)
  - Formula: Prediction = avg(Trees)
  - Use: Ensemble prediction

- K-Nearest Neighbor (`/algorithms` card 5)
  - Formula: Class = mode(neighbors)
  - Use: Local pattern matching

- Support Vector Machine (`/algorithms` card 6)
  - Formula: f(x) = sign(Σ αi*yi*K(xi,x) + b)
  - Use: Optimal separating hyperplane

**Clustering**
- K-Means (`/algorithms` card 7)
  - Formula: J = Σ ||xi - μk||²
  - Use: Unsupervised partitioning

**Probabilistic**
- Naive Bayes (`/algorithms` card 8)
  - Formula: P(C|X) = P(X|C)P(C) / P(X)
  - Use: Fast probabilistic classification

### RAG Architectures

1. **Naive RAG** (`/rag`)
   - Method: Keyword matching
   - Speed: Fast
   - Accuracy: Medium
   - Use: Simple searches

2. **Vector RAG** (`/rag`)
   - Method: Semantic embeddings
   - Speed: Moderate
   - Accuracy: High
   - Use: Semantic understanding

3. **Retrieve & Rerank** (`/rag`)
   - Method: Two-stage retrieval
   - Speed: Slower
   - Accuracy: Very High
   - Use: High-precision applications

4. **Multimodal RAG** (`/rag`)
   - Method: Text + Metadata
   - Speed: Depends on data
   - Accuracy: High
   - Use: Diverse knowledge bases
